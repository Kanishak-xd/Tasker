# Tasker

A full-stack Task Manager application

## Tech Stack

- **Frontend**: Vite + React (JavaScript)
- **Backend**: Node.js + Express
- **Styling**: Tailwind CSS v4.1
- **Database**: MongoDB Compass
- **Authentication**: JWT tokens

## Getting Started

### Frontend Setup

```bash
cd Frontend
npm install
npm run dev
```

### Backend Setup

```bash
cd Backend
npm install
npm start
```
